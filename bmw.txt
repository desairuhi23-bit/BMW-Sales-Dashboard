BMW SALES DASHBOARD (Power BI Project)

This is my first dashboard project created using Power BI.
The dashboard represents BMW sales data in a visual and interactive way.

Features:
- Total Revenue Analysis
- Total Units Sold
- Year-wise Sales Trends
- Fuel Type Distribution
- Top 5 Models by Revenue
- Sales & Units by Color
- Average Selling Price

Tools Used:
- Power BI

Purpose:
This project helps to understand business insights and improve data visualization skills.
